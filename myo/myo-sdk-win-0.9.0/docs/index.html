Just got your first Myo armband? Be sure to run the Myo Connect application on your PC or Mac computer to get set up. See the Getting Started section for more details.

Welcome to the Myo SDK. This manual is divided into the following sections:

  • Getting Started provides an introduction to the Myo armband and shows you how to get set up.
  • The Myo SDK provides an overview of the Myo SDK's structure and core concepts, and tells you how to use the SDK in your own applications.
  • Writing Myo Scripts provides a tutorial on how to use scripts to control applications via Myo Connect.
  • Myo Script API Reference provides a reference for Myo Scripts.
  • Release Notes And Known Issues provides an overview of what's new in each release as well as any known issues in the current release.

API reference documentation is also available using the navigation bar on the left.
