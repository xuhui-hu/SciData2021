Detailed Description

template<typename T>
class myo::Quaternion< T >

A quaternion that can be used to represent a rotation.

This type provides only very basic functionality to store quaternions that's sufficient to retrieve the data to be placed in a full featured quaternion type.

Examples:
hello-myo.cpp.
