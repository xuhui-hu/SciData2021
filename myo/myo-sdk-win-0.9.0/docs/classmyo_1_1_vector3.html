Detailed Description

template<typename T>
class myo::Vector3< T >

A vector of three components.

This type provides very basic functionality to store a three dimensional vector that's sufficient to retrieve the data to be placed in a full featured vector type. A few common vector operations, such as dot product and cross product, are also provided.
